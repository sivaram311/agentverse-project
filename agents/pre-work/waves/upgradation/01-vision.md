# 01 — Vision: Stable Functional Upgradation

**Initiative:** AgentVerse stable functional upgrade  
**Base:** `v0.2.2-stable` · Branch: `feature/upgradation-functionality`  
**Date:** 2026-07-15

## Goal

Make the stable office a trustworthy **operator work plane**: Session Desk findability, portal status/cancel truth, deep-link hire from ProdDeck, honest quests, thin incident land — without restyling the 3D office.

## Journeys

1. Operator finds any session (search/filter) → opens → sees live status → can cancel.
2. Home Dispatch opens `/desk?...` → AV lands with desk/hire/brief → return to Home.
3. Prompt creates a quest that tracks portal STREAMING/complete/fail — no fake %.
4. WAITING_PERMISSION / WAITING_PLAN surfaces real controls or honest Portal fallback.
5. Brief/incident strip on hire; dismiss; work continues in chat.

## Success metrics

Session findability · cancel/status match portal · deep-link hire+return · quest↔session · Realme smoke · other apps untouched on promote.
